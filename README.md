# Web Development Learning Journey

This repository is a master collection of everything I have learned and am continuing to learn in Web Development. It is structured to separate structured course work, independent technology learning, mini projects, reusable snippets, and reference notes.

## 🗂️ Repository Structure

### `01-Courses/`
Complete, structured coursework and bootcamps I have followed. The internal directory structure of these courses has been kept exactly as originally created during the learning process.
- **The-Odin-Project/**: Contains projects and exercises from the Odin curriculum.
- **Code-With-Harry/**: Contains tutorials and exercises covering HTML/CSS, JS, React, Node, etc.

### `02-Self-Learning/`
Explorations, tutorials, and scripts for individual technologies and paradigms learned outside of full structured courses. 
- Currently includes: `angular`, `docker`, `php`, `typescript`.

### `03-Mini-Projects/`
Larger, independent projects that combine knowledge from multiple places, acting as robust portfolio pieces or significant practice applications. Includes applications like the discord-clone, spotify-clone, bitlinks, etc.

### `04-Libraries-&-Concepts/`
A segregated space for specific npm packages (`libraries/`) and isolated conceptual learning instances or snippets (`concepts/`) acting as references for future projects.

### `05-Notes-&-Resources/`
A place for Markdown notes, cheat sheets, useful links, and other helpful resources for quick reference.
